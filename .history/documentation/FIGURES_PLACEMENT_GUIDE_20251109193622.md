## üé® Figure Usage Tips

### For README.md
- Use PNG format for better GitHub rendering
- Keep image width reasonable (max 1200px recommended)
- Add descriptive alt text for accessibility
- Use relative paths: `visualization/figures/filename.png`

### For Presentations
- Use PDF format for best quality when printing
- Use PNG format for digital presentations
- Ensure images are at least 1920x1080 resolution
- Consider adding figure captions below images

### For Documentation
- Use relative paths from the documentation file location
- Example: From `documentation/DA2_PPT_Content.md`, use `../visualization/figures/filename.png`
- Add figure numbers and captions for formal documentation
